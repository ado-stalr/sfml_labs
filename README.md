### Инструкция для сборки

1. Переместиться в папку проекта "sfml.x"
2. Выполнить команду 
    ```
    cmake -G "MinGW Makefiles" .
    ```
   для инициализации сборки проекта
3. выполнить команду 
    ```
    cmake --build .
    ```
   для сборки проекта
